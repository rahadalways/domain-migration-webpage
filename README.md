# Domain Migration Webpage

A modern, high-conversion **Domain Migration Webpage** designed to inform users about domain changes professionally. This template features a premium dark aesthetic with interactive elements to maintain user trust during transitions.

### 🚀 Key Features

- **Premium Dark UI:** Built with a sleek dark theme using Glassmorphism effects.
- **Dynamic Visuals:** Includes animated background orbs, a grainy noise texture for depth, and a grid overlay.
- **Interactive Experience:** Custom interactive cursor glow that follows mouse movement.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens.
- **Engagement:** Floating WhatsApp button for instant customer support.

### ⚙️ Customization Guide

Open `index.html` and update the following sections:

1.  **Domain Mapping:** Update the `.domain-flow` section with your old and new URLs.
2.  **SEO Meta Tags:** Change the `<title>`, `description`, and `keywords` in the `<head>`.
3.  **Branding:** Replace "BM Digital" with your brand name in the `.brand` div.
4.  **Contact Info:** Update the WhatsApp link and the footer email address.
5.  **Payment Graphics:** Ensure your payment method images are placed in the `Images/` folder.

### 🌐 Deployment Tip

To automatically redirect users from your old domain to the new one, ensure you configure the `.htaccess` file correctly on your old server.
